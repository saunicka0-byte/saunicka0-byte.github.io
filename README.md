# saunicka0-byte.github.io

A simple modern-looking drag & drop tier list website built with pure HTML, CSS, and JavaScript.

# ✨ Features
* Modern dark UI
* Drag & drop cards
* Upload custom images
* Responsive layout
* No frameworks
* Single-file setup

# 🖼 Adding Images

Click the + Add Image button to upload images from your device.

Uploaded images automatically become draggable tier cards.

* S Tier
* A Tier
*  B Tier
